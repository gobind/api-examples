= Wordnik + Ruby

If you'd like to hack the Wordnik API using Ruby, check out the official Wordnik API gem: http://github.com/wordnik/wordnik-ruby

Hello Dictionary is an example Rails application and tutorial to help you get familiar with the wordnik-ruby gem: http://github.com/wordnik/api-examples/tree/master/src/rails/hello_dictionary/
