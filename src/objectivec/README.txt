To compile and run this code, you need the following:

XCode 3.2 or greater
The ASI HTTP library (http://allseeing-i.com/ASIHTTPRequest/)
The JSON Framework (http://code.google.com/p/json-framework)

Follow the instructions for the ASI library and enable Reachability 2.0.  They have excellent documentation for configuring your project appropriately.

This service access layer is very rough and should serve as a guideline for your own implementation.
